# SmartSOC IRS - AI-Powered Security Operations Center

#📌 Overview

SmartSOC IRS (Incident Response System) is an AI-powered Security Operations Center platform designed to detect, analyze, and respond to cybersecurity threats in real time. The system combines machine learning, intelligent threat analysis, and live monitoring to enhance incident response workflows and improve organizational security posture.


## 🚀 Key Features

- ⚡ Real-Time Threat Monitoring Dashboard
- 🤖 AI-Powered Security Analyst Chatbot using Google Gemini API
- 🎯 Phishing Detection Model with 92% Accuracy
- 📊 Live Threat Visualization
- 🔔 Incident Alert & Response Management
- 🔐 Secure Authentication & Access Control
- 📈 High-Performance FastAPI Backend
- 🌐 REST API Integration
- 📱 Responsive React Frontend
